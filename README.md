# メモ・アイデアストックダッシュボード

散在しがちな「日々の気づき」や「ブログ・note記事のアイデア」を一つの画面で一元管理するための、Chrome拡張機能型ダッシュボードです

ブラウザ上のどこからでもショートカットキー一発で呼び出し、構造化されたメモを素早くストックできます。バックエンド（データベース）にはGoogleスプレッドシートを利用しており、サーバーレスで手軽に運用可能です

## 主な機能

- **いつでもどこでも呼び出し**: Chrome拡張機能として動作し、ショートカットキー（例: `Alt + Shift + K`）で画面上に大きめの固定ウィンドウとしてポップアップします
- **アイデアの構造化登録**: 単なるテキストメモではなく、「詳細の記載」「わからない単語と調査結果」「関連リンク」「自分の言葉で整理」「たとえ」「カテゴリ」に分けて思考を整理・記録できます
- **特殊コピペ対応**: `<メモ, URL, タイトル>` の形式でクリップボードからペーストすると、自動的にリンク情報としてパース・一覧化されます
- **入力ロック機能**: 入力中のデータはローカルストレージに自動保存されます。完了ボタンを押すまで新しい登録をロックし、誤操作によるデータ消失を防ぎつつ、一つのテーマに集中できます
- **サーバーレスDB（GAS連携）**: 登録したデータは、Google Apps Script (GAS) のWebhookを経由してGoogleスプレッドシートに保存されます

## 技術スタック

- **フロントエンド**: Next.js (App Router, Static Export), React, TypeScript, Tailwind CSS
- **バックエンド / データベース**: Google Apps Script (GAS), Google スプレッドシート
- **プラットフォーム**: Google Chrome 拡張機能 (Manifest V3)

## セットアップ手順

本システムを自身の環境で動かすための手順です。

### 1. Google スプレッドシート（データベース）の準備

1. 新しいGoogleスプレッドシートを作成します
2. 1行目のA列〜K列に以下のヘッダーを正確に入力します
   - `id`, `details`, `unknown_words`, `related_links`, `own_words`, `metaphor`, `categories`, `isUsed`, `draftUrl`, `createdAt`, `updatedAt`
3. スプレッドシートのURLから「スプレッドシートID」をコピーして控えておきます

### 2. Google Apps Script (GAS) の構築

1. スプレッドシートのメニューから「拡張機能」>「Apps Script」を選択します
2. 開いたエディタ（`コード.gs`）に、API連携用のコードを貼り付けます。
3. コード上部にある `SPREADSHEET_ID` の部分に、手順1で控えたスプレッドシートIDを設定します
4. 右上の「デプロイ」>「新しいデプロイ」を選択し、歯車アイコンから「ウェブアプリ」を選択します
5. 以下の設定でデプロイします。
   - **実行するユーザー**: 「自分」
   - **アクセスできるユーザー**: 「全員」
6. 発行された「ウェブアプリのURL」をコピーします

### 3. フロントエンドのビルド

1. プロジェクトのルートディレクトリに `.env.local` を作成し、先ほどコピーしたGASのURLを設定します。

   ```env
   NEXT_PUBLIC_GAS_WEB_APP_URL="あなたのウェブアプリのURL"
   ```

   1. 依存パッケージをインストールし、ビルドコマンドを実行します

   2. ※ビルドスクリプトにより、Chrome拡張機能でエラーとなる \_next などの予約語ファイルが自動でクリーンアップ・置換され、out フォルダが生成されます

2. Chrome拡張機能としての読み込み
   1. ChromeのURLバーに chrome://extensions/ と入力し、拡張機能の管理画面を開きます
   2. 右上の「デベロッパー モード」をONにします
   3. 左上の「パッケージ化されていない拡張機能を読み込む」ボタンをクリックし、生成された out フォルダを選択します
   4. 拡張機能「Idea Stock Dashboard」がエラーなく表示されれば成功です
   5. 左のメニューから「キーボード ショートカット」を開き、拡張機能を呼び出すためのショートカット（例: Alt + Shift + K）を設定してください
